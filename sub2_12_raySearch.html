RAY.Search

전세계 검색오픈소스 1위 제품인 ElasticSearch 검색엔진에
한국형 검색에 최적화된 다양한 기능과 개발 노하우와 묶어서
차별화된 방식으로 데이터를 탐색하고 분석하는 검색솔루션입니다.
주요 기능

  • 01대용량 검색 및 확장성

    대용량 분산처리 기술로 수평적 확산

    서비스 중단없이 검색 서버 확장 지원

  • 02자체 로드밸런싱

    클러스터 조정 하위 시스템(Discovery)

    기능 내장으로 서버 확장 및 자체 로드 밸런싱

  • 03뛰어난 검색 기능

    형태소, 어절, n-gram, 초성 등 다양한 검색 단위 지원

    불리언 검색, 와일드카드 검색, 구 단위 검색 기능 제공

    실시간 색인 업데이트 기능 제공

    20여 개 언어에 대한 다국어 검색 가능

  • 04검색 확장 기능

    통합검색, 상세검색, 검색어자동완성, 인기검색어

    첨부파일검색(MS-Office, PDF, HWP 등)등의 다양한 부가 검색기능 확장 제공

  • 05웹 기반 관리기

    검색 품질 향상을 위해 사전 등록/수정/삭제를 할 수 있는 관리기 제공

    관리 사전을 엔진에 직접 반할 수 있는 기능 제공

특장점
RAY.Search의
차별화된 서비스를 경험하세요.

기업 내 인트라넷 시스템에 산재되어 있는 축적된 지식정보를 검색을 통해 찾고, 이를 통해 새로운 지식정보가 생산되는 선순환 구조의 만들어 보십시오.

대용량 검색 및 확장성

  • 노드로 구성되는 분산시스템 구조

고가용성(HA, High Availability)

  • 데이터 원본과 복사본을 서로 다른 위치에 나누어 저장
  • 문제 발생시 데이터를 다른 노드로 옮기는 작업을 알아서 수행

멀티 테넌시(Multi-tenancy)

  • 단일 소프트웨어 인스턴스로 서로 다른 여러 사용자 그룹에 서비스를 제공할 수 있는 소프트웨어 아키텍처

RESTFul API

  • 요청/응답에 JSON을 사용해 개발 언어, 운영체제, 시스템에 관계없이 다양한 플랫폼에서 활용이 가능

다국어 검색 확장

  • 중국어, 일본어, 폴란드어, 베트남어 우크라이나어 등 오픈된 확장 플러그인을 통해 별도의 비용 없이 다국어 검색 제공

모니터링 & 관리기

  • 검색엔진 서버를 더 쉽게 운영하고 관리할 수 있는 웹관리기를 제공
