Ray.Commendation

문서중앙화/집중화를 통해 저장되거나 인트라넷 시스템에 등록되어 있는
다양한 비정형 컨텐츠를 활용하여 새로운 지식을 생산할 수 있는 검색 솔루션을 경험해보세요.
주요 기능
  • 검색 인터페이스 제공

    윈도우 탐색기/웹 검색 결과를
    동시 조회하여 사용자
    지식정보 활용 가능

  • 서식 문서 디지털화

    정형 및 비정형 데이터에서 필요 추출
    정보 선별 및 데이터 변환 후
    구조화 하여 적재

  • AI 모델링 최적화 지능형 검색

    사용자 질의 의도를 의미적으로 이해하고
    시스템에 산재된 데이터 내에서
    하나의 정답만을 제공

특장점
Ray.Commendation의
차별화된 서비스를 경험하세요.

Ray.Commendation는 업무 데이터 중앙 집중화를 통해
기업 정보의 유출을 막고 안전하고 효율적인 업무 관리 환경을
제공하는 레이풀의 문서중앙화 보안 솔루션 입니다.

자연어 검색 지원

  • - ’휴가’ 또는 ‘백신 접종 시 휴가 신청’ 같은
    의도에 정확한 질문으로 답변 탐색이 가능

자동분류 기능 지원

  • - Paraphrasing, ETL, TA 기술을 활용하여
    고객 의도분석으로 다양한 형태 분석 서비스 제공

다수의 검색 엔진 사용

  • - 데이터 유형에 따른 적합한 검색엔진 사용으로
    최적의 응답을 선정 (MRC, FAQ, Table QA, Passage Search Engine)

다양한 문서 형식 지원

  • - PDF, DOCX, XLS, HWP 형식의
    정형/비정형 문서 업로드 기능 지원
