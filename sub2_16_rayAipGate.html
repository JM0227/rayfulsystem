RAY.AIP Gateway

MS365를 활용하는 기업이 늘어나면서 기존에 운용 중인 DRM과의 연동이 필요하게 되었습니다.
DRM으로 암호화된 문서를 암호화 해제 후 AIP로 암호화하거나,
AIP로 암호화된 문서를 해제 후 DRM으로 암호화하거나,
사용자 PC에서 발생하는 모든 주요 문서를 AIP로 암호화는 보안 적용 플랫폼을 적용하십시오.
주요 기능

  • 01DRM to AIP

    DRM으로 암호화된 문서를 복호화하여 암호화를 제거하거나 AIP로 암호화

  • 02AIP to DRM

    AIP로 암호화된 문서를 복호화하여 암호화를 제거하거나 DRM으로 암호화

  • 03AIP PC Agent

    사용자 PC에서 발생하는 문서를 AIP로 암호화

특장점
RAY.AIP Gateway의
차별화된 서비스를 경험하세요.

DRM과 AIP간의 양방향 암복호화 및 로컬 PC에서 생성되는 주요 문서에 대한 AIP를 통한 암호화를 적용할 수 있는 보안 플랫폼을 도입하여 보안을 적용해보십시오.

양방향 암복호화

  • DRM과 AIP간 양방향 암복호화 지원

AIP를 통한 보안 적용

  • 사용자 PC에서 생성되는 주요문서들의 AIP를 통한 암호화로 보안성 강화
  • AIP 적용에 따른 외부 문서 전송 추적
